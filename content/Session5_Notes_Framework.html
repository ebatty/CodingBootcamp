Session 5 Notes Framework#

Python Data Types#

Objects#

Each data type is a class.

A class is a blueprint which you can use to create objects, each object is an instance of a class.

Object attributes:

  • Data attribute: the value for that specific object

  • Method: a function attached to an object. Each type of object has its own methods.

Dictionaries#

Dictionaries are unordered pairs of keys and values

Dictionary keys can be any immutable data (integers, floats, strings or Booleans). Dictionary values can be pretty much any data type.

Unordered so no indexing
