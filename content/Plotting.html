Plotting#

import numpy as np
import matplotlib.pyplot as plt

matplotlib is the most widely used plotting library in Python (although definitely not the only one!)

Styles of plotting#

There are two “styles” for plotting in matplotlib, outlined below.

Style 1: Functional programming aka MATLAB-style#

In this style, you use a bunch of commands to make and customize your plots.

plt.plot(np.arange(3))
plt.xlabel('x label')
plt.ylabel('y label')

Text(0, 0.5, 'y label')

plt.subplot(1, 2, 1)
plt.plot(np.arange(5))

plt.subplot(1, 2, 2)
plt.plot(np.arange(5))

[<matplotlib.lines.Line2D at 0x7fc83c31e510>]

Style 2: Object Oriented Interface#

In this style, the axes of different subplots are objects whose attributes you can change. This means you can easily reference subplots out of order, and it takes less code to set some of the customization.

fig, ax = plt.subplots()

ax.plot(np.arange(5))

ax.set(xlabel = 'x label',
       ylabel = 'y label');

Plotting basics#

See below for how to add axis labels, a title, where the ticks on x/y axes should occur, how to customize the tick labels, and change the axis limits. Using the second style from above, you can do all of this with set instead of needing separate commands for all.

fig, ax = plt.subplots()

data = np.arange(3)
ax.plot([0, 1, 2], data)

ax.set(xlabel = 'x',
       ylabel = 'y',
       title = 'title',
       xticks = [.5, 1.5],
       xticklabels = ['Monday', 'Friday'],
       ylim = [0, .2]);

Changing line properties#

fig, ax = plt.subplots()

data = np.arange(3)

# You can specify all of the customization of line properties with separate arguments for each one
#ax.plot([0, 1, 2], data, linestyle = '--', marker = 'o', color = 'r', linewidth = 3)

# You can also use a shortcut where you can specify line style, marker style, and color easily
ax.plot(data, '--or')

# You can also use any color by specifying hex codes
# ax.plot(data, '-o', color = '#1EC2A0')

[<matplotlib.lines.Line2D at 0x7fc83c7b31d0>]

Add legend#

# Option 1: specify after plotting

fig, ax = plt.subplots()

ax.plot(data, '-ok')
ax.plot(data*2, '-og')

ax.legend(['Label 1', 'Label 2'])

<matplotlib.legend.Legend at 0x7fc83c29fd50>

# Option 2: specify while plotting
fig, ax = plt.subplots()

ax.plot(data, '-ok', label = 'Label 1')
ax.plot(data*2, '-og', label = 'Label 2')

ax.legend()

<matplotlib.legend.Legend at 0x7fc83c04d790>

Change figure size#

fig, ax = plt.subplots(figsize = (20, 10))
ax.plot(data)

[<matplotlib.lines.Line2D at 0x7fc835f45e90>]

Save figures#

fig, ax = plt.subplots()
fig.savefig('test.pdf')

Histograms#

fig, ax = plt.subplots()

y = np.random.randn(100, 10)
reshaped_y = np.reshape(y, (-1,))
ax.hist(reshaped_y, bins = 100);

Subplots#

Standard subplots#

fig, axes = plt.subplots(nrows = 3, ncols = 2, figsize = (10, 5))

axes[0][1].set_title('hello')

plt.tight_layout()

If subplot layout still doesn’t look nice with tight_layout, try changing the proportions of the fig size to suit the plots

How do you create 6 rows and 4 columns of subplots and ensure that they’re all square?

fig, ax = plt.subplots(6, 4, figsize = (4, 6))

Gridspec#

GridSpec is useful for more complicated subplot arrangements

fig = plt.figure()

grid = fig.add_gridspec(3, 3)

ax1 = fig.add_subplot(grid[:, 0])

ax2 = fig.add_subplot(grid[0, 1:])
